# Pact + .Net example project

### Base flow


**Generate pact file (Consumer side)**

> cd Consumer
> dotnet test

**Validate pact file (Provider side)**

> cd Provider
> dotnet test

## Tools

 * .NET Core 3.1
 * XUnit
 * ASP.NET Core Web API
 * Pact .NET Core v3
